# Fabrication Journey

**Live site: https://keplrx.github.io/dfab-journal/**

Weekly journal for EP1000 — Digital Fabrication & Prototyping Fundamentals at Singapore Polytechnic (AY2026, Y2 Sem 1), kept as part of the module's continual assessment.

Covers the semester week by week: Fusion 360 CAD exercises (weeks 3–4), 3D printing (week 5), the CA1 project — a printed housing for the ESP32-C3 with its design iterations and failures — and laser cutter certification at FabLab (week 11).

Built as a single static page, hosted on GitHub Pages.
